# Smart Expense Manager 

A modern, responsive **Expense Management Web Application** built using **HTML, CSS, and JavaScript**.  
This project helps users track income and expenses with categories, wallets, charts, and a clean UI.

---

## Features

- Add income and expense transactions
- Category-based transactions (Food, Rent, Travel, etc.)
- Multiple wallets (Cash, Bank, UPI)
- Income vs Expense chart (Chart.js)
- Search transactions
- Delete transactions
- Dark mode toggle
- Data persistence using `localStorage`
- Fully responsive UI

---

## Tech Stack

- **HTML5**
- **CSS3** (Modern UI, Flexbox)
- **JavaScript (ES6)**
- **Chart.js**
- **Browser localStorage**

## How Authentication Works (Demo)

- Each transaction is stored as an object with a unique ID
- Data is saved in `localStorage` to persist after refresh
- Charts update dynamically when transactions change
- UI re-renders on add/delete actions
